##### Clone

复制远程仓库到本地

```bash
git clone <url>
```

##### Add

添加文件到本地

```bash
git add <file_name>
git add .
```

##### Commit

保存本地修改

```bash
git commit -m "message"
```

##### Push

本地上传修改到远程

```bash
git push origin master
```

##### Status

查看状态

```bash
git status
```

##### Pull

更新远程仓库

```bash
git pull origin master
```
